# Stillward — Claude Code Instructions

## Commit Rules

When creating commits:

1. **Language**: Always English
2. **No attribution**: Never add `Co-Authored-By` or `Generated with Claude Code`
3. **Atomic commits**: Group by functionality, separate files logically
4. **Message style**: Short, factual, imperative mood
5. **Format**: `<type>: <short description>`

Types: `feat`, `fix`, `refactor`, `style`, `docs`, `test`, `chore`

See `cookbooks/COMMIT_CONVENTIONS.md` for full guidelines.

## Project Structure

- `cookbooks/` — Documentation and specs
- Laravel + Vue + Inertia stack
- Sail for Docker development

## Code Style

- PHP: PSR-12, PHPStan level 8
- Vue: Composition API, ESLint + Prettier
- CSS: Tailwind CSS
